你的数学救星来啦——微积分讲座


这里将为你抖落在学习微积分与数学分析中遇到的困惑，这里将为你点明数学基础的重要性，这里将有无数优秀的学习方法同你分享，这里，将有大佬学长吴天，田野为你指点迷津，这里更有权威老师程艺为你变幻数学的魔术，这里，东区水上报告厅，一场数学的盛宴，一场属于微积分和数学分析的狂欢，即将拉开帷幕。


讲座时间

第一场 11.23 晚上19：00 微积分/数学分析B

第二场 12.2 晚上19：00 数学分析A


讲座地点

东区水上报告厅


让我们一睹本次讲座主讲人的风采吧(๑•ᴗ•๑)



程艺老师：主要从事“非线性科学”领域中若干数学理论的研究与应用工作。1987年获英国曼彻斯特大学理学博士学位。曾任中国科学技术大学数学系主任，中国科学技术大学副校长，安徽省委教育工委书记、省教育厅厅长、党组书记。



田野学长：2015级少年班学院统计学方向，曾获2016、2018年国家奖学金，2017年优秀学生奖学金（金奖），2016年全国大学生数学竞赛省一等奖，2018年暑期赴美国哥伦比亚大学参与暑期科研，热爱数学和编程。



吴天学长：2015级理科试验1班，基础数学专业。2016年获第八届全国大学生数学竞赛(数学类)预赛安徽赛区一等奖，并于次年3月获第八届全国大学生数学竞赛(数学类)决赛全国一等奖。

曾任多门数学课程助教，任数分(B)助教期间多次参与校正数分教材。第三届科大校园吉尼斯中创造速算项目校园吉尼斯纪录。
