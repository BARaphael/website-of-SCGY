权益部

我们的职责

    维护学生的合法权益：同学们在学习生活中遇到各种问题都可以向我们部门反映，我们会积极与学校后勤部门联系，解决同学们的问题。

我们的工作

    日常维权：

1.学校教学楼洗手间内能不能配备洗手液啊？

2.学校二教网络怎么这么差?

以及其他在学习生活中常见的问题。

    失物招领：

    我们会每天在食堂等地巡查，将同学们丢失的物品在平台上通报，让失物尽快归还原主。

我们的功绩

    除了这些日常维权、失物招领工作，我们还会不定期举办一些活动呢。

    例如，我们最近就做了一个回收毕业班学长的旧书的活动，将这些书留给学弟学妹们。

    我们还会不定期通过微信公众号，给大家推送一些精彩的文章，例如我们曾推送过学校各部门的详细探寻3.15时推送过防骗指南

你问我答

    Q:如果加入权益部，我要干啥呢？

    A:我们会展幵日常维权和失物招领的工作。有同学向我们反映问题后，我们便会和有关部门取得联系，展幵协商，尽快解决同学们的问题。失物招领这块，我们会定期去食堂失物招领处，拍下同学们丢失的物品，发到校权益平台上。如果大家有什么good idea，也欢迎大家提出，我们一同去实现。

    Q:平时工作会不会很累啊，会不会很影响学习啊？

    A:我们平时的工作还是很轻松的，不会耽误大家太多时间。加入我们，还可以认识学生会的很多优秀的学长学姐哦！
